## Getting Started

Prepare app:

1. 

```bash
npm install
```

2. Create `.env` file in root path and add constants (look `.env.example`)

## Development

```bash
npm run dev
```

App run on [http://localhost:3000](http://localhost:3000)

## Production

```bash
npm run build

npm run start
```

App run on [http://localhost:3000](http://localhost:3000)

## Test

```bash
npm run test
```
